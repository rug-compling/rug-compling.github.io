RuG-CompLing on GitHub

About RuG-CompLing

RuG-CompLing is an acronym for Computational Linguistics, University of Groningen.

This page shows a selection of our repositories on GitHub. For a complete list, visit GitHub.

Alpino

Alpino is a dependency parser for Dutch. See the project homepage.

Repositories related to Alpino:

  • alpino-api — API for an Alpino server
  • alpino-docker — Alpino in Docker
  • alpinocorpus — Library for handling Alpino corpora, with bindings in Go, Python and Ruby
  • alpinods — Type definitions for Go
  • AlpinoGraph — Using Cypher to query Alpino copora, also in Docker
  • dact — Decaffeinated Alpino Corpus Tool
  • paqu — Parse & Query, also in Docker

Universal Dependencies

Universal Dependencies (UD) is a framework for consistent annotation of grammar.

Repositories related to Universal Dependencies:

  • alud — A Go package for deriving Universal Dependencies from Dutch sentences parsed with Alpino
  • conllu-viewer — A web-based viewer for documents in the CoNLL-U format

More language technology

  • epar — A Java implementation of Zhang and Clark (2011)'s shift-reduce CCG parser
  • errormining — Error mining in parsing results
  • hmm-reps — Hidden Markov models for word representations

Retired projects

  • align2tex [2016] — Creates LaTeX source from word alignments
  • bimu [2016] — Bilingual Learning of Multi-sense Embeddings with Discrete Autoencoders
  • dep-brown-cluster [2014] — Syntactic extension of Brown et al. 1992 clustering algorithm
  • dep-brown-data [2014] — Induced clusters with experimental details
  • featuresqueeze [2010] — Maximum entropy and correlation-based feature selection
  • flan [2010] — RESTful Fluency Annotation client/server
  • fluency-treebank [2010] — Fluency annotations
  • glad [2015] — Groningen Lightweight Authorship Detection
  • qtleap [2017] — with a moses based translater between Dutch and English for the QTleap project
