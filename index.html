Alpino

Alpino is a dependency parser for Dutch. See the project homepage.

Some repositories related to Alpino:

  • alpino-api — API for an Alpino server
  • alpino-docker — Alpino in Docker
  • alpinocorpus — Library for handling Alpino corpora
  • alud — A Go package for deriving Universal Dependencies from Dutch sentences parsed with Alpino
  • dact — Decaffeinated Alpino Corpus Tool
  • paqu — Parse & Query
  • paqu-docker — PaQu in Docker
